## Scope

Cryogenic transfer lines (LH₂) from vacuum-insulated tank to the conditioning
stage. Vacuum-jacketed piping, cryogenic shut-off valves, thermal-contraction
joints, and venturi flow elements suitable for 20 K service.

### Interfaces

- Upstream: `28-10-20-lh2-vacuum-insulated-tank/`
- Downstream: ATA 47-40 conversion-stack inlet
- Thermal: `28-60-20-pre-cool-and-conditioning/`
