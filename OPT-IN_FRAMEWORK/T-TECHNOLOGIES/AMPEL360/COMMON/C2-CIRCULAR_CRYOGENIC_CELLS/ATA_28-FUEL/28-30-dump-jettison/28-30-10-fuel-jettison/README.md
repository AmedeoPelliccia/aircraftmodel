# ATA 28-30-10 — Fuel Jettison

| Field | Value |
|---|---|
| **ATA-100 Slot** | 28-30 Dump / Jettison |
| **PCT** | PCT-28-WTW |
| **Status** | ACTIVE (where applicable) |

## Scope

Conventional fuel jettison overboard for the WTW architecture, where
applicable per CS-25 weight-and-balance and emergency landing requirements.

> **Note:** Boil-off recovery and venting for the LH₂ path is **not** a
> jettison function. It is treated as a thermal-management function and
> lives under `28-60-30-boil-off-recovery-routing/`. This is a deliberate
> deviation from the pre-reconciliation layout, justified in BREX as a
> semantic correction (vent/recycle ≠ controlled discard).
