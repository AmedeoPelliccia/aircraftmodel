# ATA 28-10-10 — Conventional Tank

| Field | Value |
|---|---|
| **ATA-100 Slot** | 28-10 Storage |
| **PCT** | PCT-28-WTW |
| **Status** | ACTIVE |

## Scope

Conventional fuel reservoir applicable to the **WTW near-term certifiable
architecture** when SAF or Jet-A is carried as a transitional or back-up
energy source.

Includes:

- Cylindrical / kidney-pod metallic vessel
- Bladder cells (if applicable)
- Vent / fill / drain fittings per CS-25 Subpart E
- Quantity probes (capacitive)
