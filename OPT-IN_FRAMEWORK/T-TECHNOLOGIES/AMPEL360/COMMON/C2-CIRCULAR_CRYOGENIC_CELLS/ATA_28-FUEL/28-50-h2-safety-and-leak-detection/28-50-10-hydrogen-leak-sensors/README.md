# ATA 28-50-10 — Hydrogen Leak Sensors

| Field | Value |
|---|---|
| **PCT** | PCT-28-COMMON (any H₂ path) |
| **Status** | ACTIVE |

## Scope

Distributed H₂ leak sensor network. Technology shortlist:

- Catalytic bead (mature, proven)
- Semiconductor (MOS / SnO₂)
- Fibre-optic (intrinsically safe, distributed sensing)

Target detection response: ≤ 5 s. Redundancy ≥ 2 dissimilar channels.
