# ATA 28-60-20 — Pre-Cool and Conditioning

| Field | Value |
|---|---|
| **PCT** | PCT-28-BWB |
| **Status** | ACTIVE |

## Scope

Pre-cool of feed lines and conditioning of PM stream into the conversion
stack inlet:

- Cooldown sequence prior to first LH₂ flow
- Heat-exchanger network (vaporiser, recuperator)
- Temperature ramp control to avoid thermal shock
- Coupling with `28-20-30-nh3-cracker-h2-feed/` if NH₃ path active
