# ATA 28-60-10 — Vacuum Jacket Monitoring

| Field | Value |
|---|---|
| **PCT** | PCT-28-BWB |
| **Status** | ACTIVE |

## Scope

Continuous monitoring of the vacuum-insulated tank jacket integrity:

- Vacuum-pressure transducer in jacket annulus
- MLI degradation indicators (long-term boil-off trend)
- Heat-leak budget tracking against design baseline
