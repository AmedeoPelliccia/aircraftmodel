# ATA 28-40-10 — Quantity Indicating

| Field | Value |
|---|---|
| **ATA-100 Slot** | 28-40 Indicating |
| **PCT** | PCT-28-COMMON (BWB + WTW) |
| **Status** | ACTIVE |

## Scope

Common quantity-gauging architecture for stored prime material mass `M_i`.

- Capacitive / ultrasonic / optical level sensing
- Mass calculation from level + density (T, P corrected)
- Crew display (EICAS / CAS) — fuel quantity gauge
- Sensor redundancy (≥ 2 dissimilar channels)

Target accuracy: ≤ 1.0 % full-scale.
