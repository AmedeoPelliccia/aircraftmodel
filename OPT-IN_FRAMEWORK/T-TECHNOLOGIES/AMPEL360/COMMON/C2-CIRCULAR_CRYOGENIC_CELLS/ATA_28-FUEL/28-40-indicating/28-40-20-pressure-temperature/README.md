# ATA 28-40-20 — Pressure / Temperature Indicating

| Field | Value |
|---|---|
| **ATA-100 Slot** | 28-40 Indicating |
| **PCT** | PCT-28-COMMON (BWB + WTW — H₂ extended range) |
| **Status** | ACTIVE |

## Scope

State-vector instrumentation for `P_i` (pressure) and `T_i` (thermal state).

- Cryogenic-rated temperature sensors (PT-100, Si-diode) — 20 K to 350 K range
- Pressure transducers (absolute and differential)
- Vent-line temperature monitoring (thermal plume detection)
- Display: pressure indicator, temperature indicator, phase status lamp

For LH₂ (BWB) the operating range extends to cryogenic; for conventional
fuel (WTW) the range is ambient. Sensor selection differs but the indicating
function is common.
