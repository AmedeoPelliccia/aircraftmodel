## Scope

This sub-chapter covers the design, integration, and certification of prime
material reservoirs (cryogenic tanks) within the C² CELL architecture.

### Key Topics

- LH₂ reservoir geometry (conformal bay / cylindrical pod per variant)
- Cryo-insulation system (MLI, VCS, foam)
- Structural mounting and cryo-thermal interface (ATA 51)
- Fill / vent / drain ground interfaces
- Boil-off management and pressure relief
- NH₃ containment (double-wall, pressurised vessel — WTW variant)
- LN₂ blanket reservoir

### Design Drivers

| Driver | Metric | Target |
|---|---|---|
| Gravimetric index | kg_PM / kg_system | ≥ 0.40 |
| Boil-off rate | % mass / hour | ≤ 0.05 |
| Fill / drain time | minutes | ≤ 45 |
