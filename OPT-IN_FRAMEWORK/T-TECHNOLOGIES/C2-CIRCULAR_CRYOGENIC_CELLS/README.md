# C2-CIRCULAR_CRYOGENIC_CELLS

Circular Cryogenic Cell technologies for sustainable aviation fuel systems.

## Overview

This section contains specifications and designs for cryogenic fuel cells that support:
- LH2 (Liquid Hydrogen)
- NH3 (Ammonia)
- N2 (Nitrogen)
