# ATA 44 BWB — Requirements (ΔREQ)

**Parent:** [BWB README](../README.md)

## Scope

BWB-only ATA 44 Cabin Systems requirements that deviate from or are not present
in the SHARED baseline. Structured per ATA 44 subchapter.

## Files

| File | Subchapter | Subject |
|------|-----------|---------|
| `REQ_BWB_44_000_GENERAL.yaml` | 44-000 | BWB-specific applicability and scope |
| `REQ_BWB_44_100_CABIN_CORE.yaml` | 44-10 | Distributed ZC, bay segmentation, BBU, zone topology |
| `REQ_BWB_44_200_IFE.yaml` | 44-20 | Per-bay IFE Bay Node requirements |
| `REQ_BWB_44_300_EXTERNAL_COMM.yaml` | 44-30 | External comm antenna placement (BWB fuselage) |
| `REQ_BWB_44_500_CABIN_MONITORING.yaml` | 44-50 | Bay-partitioned monitoring, smoke detection per bay |
| `REQ_BWB_44_600_MISC.yaml` | 44-60 | BWB miscellaneous system deltas |
