## Scope
This directory holds the WTW wing maintenance plan registry hook for AMPEL360 Q100.
It provides classification of wing repair tasks, spare parts, supplier responsibility,
and material circularity tier, consistent with the SSOT maintenance plan classification
standard defined under `I-INFRASTRUCTURES/.../STD_MAINT_PLAN_CLASSIFICATION/`.
