# ANALYSIS — WTW Cargo Restraint (CS-25.787/789/561)

**Evidence ID:** EV-WTW-50-002  
**Type:** Analysis  
**Programme:** AMPEL360 WTW  
**Regulation:** CS-25.787 / CS-25.789 / CS-25.561  
**Delta:** DELTA-WTW-50-006  
**Status:** STUB — LC01. Full analysis gate LC05.

---

## 1. Objective

Demonstrate that WTW cargo net/restraint attach fittings and cargo floor structure
sustain the load cases per CS-25.787 (stowage compartment strength), CS-25.789
(retention of items of mass) and CS-25.561 general load factors.

---

## 2. Method (Planned)

- Derive WTW-specific cargo restraint load cases from aircraft loads envelope
- FEA of belly-hold floor, keel beam and restraint fitting attachments
- Verify restraint fitting margins at ultimate load per CS-25.787

---

## 3. Acceptance Criteria

- Cargo net fitting: MS ≥ 0.0 at ultimate
- Floor structure: no permanent deformation at limit load
- Restraint system retains all items per CS-25.789 inertia loads
