# INSPECTION — WTW Drain & Vent Installations (CS-25.1001/1103)

**Evidence ID:** EV-WTW-50-005  
**Type:** Inspection  
**Programme:** AMPEL360 WTW  
**Regulation:** CS-25.1001 / CS-25.1103  
**Delta:** DELTA-WTW-50-004  
**Status:** STUB — LC01. Inspection plan gate LC04.

---

## 1. Objective

Demonstrate that WTW drain mast and vent port installations comply with
CS-25.1001 (drainage) and CS-25.1103 (ventilation) requirements.

---

## 2. Inspection Scope (Planned)

- Verify drain mast penetration coordinates against ICD-WTW-50-004
- Measure Class B sealant fillet width at each penetration
- Pressure test drain fitting per installation limit (pressurisation cycle)
- Measure vent port flow rate per CS-25.1103 minimum requirement (TBD l/min, LC04)
- Verify no fluid accumulation ≥ limit in belly hold after drain test

---

## 3. Acceptance Criteria

- Drain mast coordinates within ± TBD mm of ICD-WTW-50-004 values
- Sealant fillet width ≥ minimum per installation limits
- Vent flow rate ≥ CS-25.1103 minimum (TBD l/min, LC04)
- No fluid accumulation ≥ limit per WTW-50-400-001
