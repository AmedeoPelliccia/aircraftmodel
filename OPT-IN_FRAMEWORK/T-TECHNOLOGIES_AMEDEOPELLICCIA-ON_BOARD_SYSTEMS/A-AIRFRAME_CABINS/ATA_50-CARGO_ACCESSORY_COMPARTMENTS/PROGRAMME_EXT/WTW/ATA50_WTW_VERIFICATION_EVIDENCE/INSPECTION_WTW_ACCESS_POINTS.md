# INSPECTION — WTW Cargo Door Access Points (CS-25.1411)

**Evidence ID:** EV-WTW-50-004  
**Type:** Inspection  
**Programme:** AMPEL360 WTW  
**Regulation:** CS-25.1411 Required safety equipment — accessibility  
**Delta:** DELTA-WTW-50-002  
**Status:** STUB — LC01. Inspection plan gate LC04.

---

## 1. Objective

Demonstrate that WTW cargo door clear-opening dimensions satisfy CS-25.1411
access requirements for required safety equipment.

---

## 2. Inspection Scope (Planned)

- Dimensional inspection of FWD and AFT cargo door clear-opening width and height
- Verify sill height above ground line per ICD-WTW-50-002
- Functional test of door opening sequence; record actuation time
- Verify ground locking pin engagement depth per ICD-WTW-50-002

---

## 3. Acceptance Criteria

- Clear opening ≥ values defined in ICD-WTW-50-002 (TBD at LC03)
- Actuation time ≤ TBD s (LC04)
- Locking pin engagement ≥ minimum per ICD-WTW-50-002
