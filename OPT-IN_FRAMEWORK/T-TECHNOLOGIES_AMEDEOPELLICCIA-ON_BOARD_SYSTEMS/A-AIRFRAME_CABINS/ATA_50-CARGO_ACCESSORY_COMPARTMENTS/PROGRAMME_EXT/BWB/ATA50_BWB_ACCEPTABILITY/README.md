# ATA 50 BWB — Acceptability

This directory contains BWB-only acceptability limits for ATA 50
Cargo & Accessory Compartments.

## Contents

| File | Subject |
|---|---|
| `ACC_BWB_INSTALLATION_LIMITS.md` | Installation limits (liner panels, drain fittings, door latches) |
| `ACC_BWB_DAMAGE_LIMITS.md` | In-service damage limits for BWB cargo compartment components |
| `ACC_BWB_REWORK_RULES.md` | Rework criteria cross-linked to ATA 20 standard practices |
| `ACC_BWB_CLEARANCES_AND_ACCESS.md` | BWB-only: spar bay clearance and maintenance access limits |

## Rule

All acceptability limits cross-reference ATA 20 SHARED standard practices as the
baseline. This directory provides only BWB-specific deltas from that baseline.
Numerical limits marked TBD are locked at LC03–LC05.
