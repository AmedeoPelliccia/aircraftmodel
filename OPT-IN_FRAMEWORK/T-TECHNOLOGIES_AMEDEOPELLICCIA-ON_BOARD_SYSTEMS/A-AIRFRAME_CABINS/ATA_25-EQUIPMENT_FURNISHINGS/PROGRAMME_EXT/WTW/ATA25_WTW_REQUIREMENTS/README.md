## Scope

WTW-only requirement packs by ATA 25 subchapter. These supplements add WTW-specific
constraints on top of the SHARED ATA 25 baseline. Requirements shall not duplicate
SHARED content; they add, qualify, or restrict SHARED requirements for the WTW variant.
