## Scope

ATA 51 is the **structural standard practices** chapter for the AMPEL360 Q100. It owns all
shared technical content for materials, fasteners, surface treatments, repair acceptability,
inspection/NDT selection, and bonding integrity verification.

ATA 20 (`ATA_20-00-00_STANDARD_PRACTICES_AIRFRAME_SHARED/`) references this chapter for
governance routing. Local area implementations (ATA 53, ATA 57) overlay and extend this baseline.
