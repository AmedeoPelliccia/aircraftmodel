## Scope

Engine fuel and control system interfacing LH₂ delivery (from ATA 28 C² CELL) with FADEC. Configuration-specific: flow rates, pressure schedules, and FADEC algorithms differ between WTW (with NH₃ cracking) and BWB (LH₂-only PEM).
