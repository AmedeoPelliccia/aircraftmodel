## Scope

Exhaust system design and emissions monitoring. Configuration-specific: WTW turbofan exhaust includes NOₓ and H₂O management; BWB fuel cell exhaust is primarily H₂O. Both must meet ≤5% CAEP/8 NOₓ target.
