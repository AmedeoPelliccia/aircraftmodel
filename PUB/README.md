# PUB — Publications & Deliverables

## Structure

| Directory | Content |
|---|---|
| `CSDB/BREX/` | Project Business Rules Exchange (S1000D) |
| `CSDB/DM/` | S1000D Data Modules |
| `EXPORT/` | Formatted deliverables (PDF, HTML, DOCX) |

## Rules

- `CSDB/DM/` contains only validated S1000D XML data modules
- `EXPORT/` contains generated outputs — rebuild from SSOT source
- Never place raw engineering evidence here (use `SSOT/` instead)
