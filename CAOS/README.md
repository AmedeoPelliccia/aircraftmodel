# CAOS — Continuous Airworthiness & Operations Sustainment

## ID-A360-Q100

This directory contains artifacts related to the continuous airworthiness
and operational sustainment of the ID-A360-Q100 aircraft.

### Scope

- Airworthiness Limitation Items (ALIs) for cryo systems (KNOT-ATA04-00-00-001)
- Maintenance planning documents
- Operational procedures
- Service bulletins and airworthiness directives

### Status

CAOS artifacts will be populated as the program progresses through LC10–LC11.
